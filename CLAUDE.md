# 프로젝트 규칙

## 기술 스택
- HTML + CSS + 바닐라 JS
- 외부 라이브러리 사용 금지

## 디자인 규칙
- 색상: #faa3ab (살구), #F3F4F6 (배경)
- 폰트: 시스템 기본 폰트
- 모바일 우선

## 금지 사항
- React, Vue 등 프레임워크 사용 금지
- npm, 빌드 도구 사용 금지

## 보안 경계
- API 키는 반드시 환경변수로. 코드에 하드코딩 금지
- 사용자 입력은 항상 검증

## 품질 기준
- 페이지 로딩 3초 이내
- 에러 발생 시 사용자에게 안내 메시지 표시

## 금지 패턴
- console.log 디버깅 코드 남기지 않기
- any 타입 사용 금지 (TypeScript일 때)
- 한 파일에 300줄 넘기지 않기
